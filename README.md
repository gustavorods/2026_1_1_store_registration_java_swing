# Store Registration Form Java Swing

**Tagline**: A simple Java Swing app to register customers in a store.

## Description
This application uses Java Swing to provide a graphical interface for customer registration. Through a form built with JFrame, users can enter information such as name and age, allowing the system to register customers in the store in a simple and practical way.

## Installation
```bash
git clone https://github.com/gustavorods/2026_1_1_store_registration_java_swing.git
cd 2026_1_1_store_registration_java_swing
```

## Usage
To run the application in VSCODE, execute the following command in the terminal:
```bash
javac Cadastro.java && java Cadastro
```
